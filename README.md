Foodian is a premium food discovery and management platform built with Next.js, designed for exploring, adding, and managing curated recipes. It features secure authentication, protected routes, and seamless MongoDB integration for dynamic food management. With a modern UI, smooth interactions, and optimized performance, Foodian delivers a refined gourmet experience for food lovers.

used technology:
Next.js, react icons, tailwind css, mongodb, next-auth, react-hot-toast
